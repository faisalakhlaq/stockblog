Welcome to the Stock & Crypto Currency Data Fetcher.

Stock Data Fetcher

Its a simple hobby project which consists of two parts. Users can fetch any given stock's data from yahoo for given dates. Currently there are just 100 stocks to choose from. Data is then displayed in a candlestick chart. One page is used for providing general guidelines about stocks terms. There is an admin panel for adding, updating and deleting guidelines and material for the website. A user module is also developed to take care of user registration, login, logout, password recovery by email and other user related operations, in-case the admins forget passwords. Data about stocks and users is saved in a PostgreSQL database.
Tools used are:
Python, Bokeh Flask (flask_login, forms, SQLAlchemy, Blueprint) and PostgreSQL.

Crypto Currency Portfolio

The Crypto Currency Portfolio web app lets the user to enter their currencies and amount of each currency and fetches respective currency data from www.coinmarketcap.com website. The data is fetched by making API calls to coinmarketcap. Users portfolio is displayed in a tabular form afterwards.
